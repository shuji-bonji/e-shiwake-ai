# サンプルプロンプト集

e-shiwake MCP サーバーに接続した状態で、以下のプロンプトをコピペして試せます。
AIエージェント上で仕訳入力から帳簿確認まで、UIなしで完結できます。

## 日常の経費記帳

```
今日、Amazonで3,980円のUSBケーブルを購入しました。
クレジットカード払いです。仕訳を作成してください。
```

```
1月15日にJR東日本で電車代1,200円を現金で払いました。仕訳を登録して。
```

```
今月のさくらインターネットのサーバー代1,320円が普通預金から引き落とされました。
消費税10%の課税仕入です。
```

## 家事按分（複合仕訳）

```
1月20日のNTTドコモの携帯電話代10,000円を記帳してください。
事業用80%、プライベート20%で家事按分します。普通預金から引き落とし。
```

## 売上の記帳

```
1月31日にクライアントAから売上500,000円が普通預金に入金されました。
売掛金の回収です。仕訳を作成してください。
```

```
2月分のシステム開発費として、クライアントBに売掛金300,000円を計上してください。
消費税10%です。
```

## 源泉徴収の記帳（複合仕訳）

```
3月分のコンサルティング報酬110,000円（税込）をクライアントCに請求しました。
源泉徴収税10,210円が差し引かれ、普通預金に99,790円入金されました。
仕訳を作成してください。
```

## 帳簿の確認

```
2025年度の試算表を見せてください。貸借が一致しているか確認したいです。
```

```
2025年度の損益計算書を表示して。今年の利益はどのくらい？
```

```
2025年度の貸借対照表を出してください。
```

```
2025年度の消費税集計を見せて。納付税額はいくらになりそう？
```

## まとめて確認（確定申告準備）

```
2025年度の全帳簿（試算表、損益計算書、貸借対照表、消費税集計）を
まとめて表示してください。確定申告の準備です。
```

## 仕訳の一覧と検索

```
2025年度の仕訳を全部見せてください。
```

```
2025年度にAmazonとの取引はありますか？一覧で見せて。
```

## 応用：月次ルーティン

以下のように、複数の操作をまとめて依頼することもできます。

```
1月の経費を記帳します。以下をまとめて仕訳作成してください。

- 1/5  さくらインターネット サーバー代 1,320円 普通預金
- 1/10 Amazon USBハブ 2,980円 クレジットカード
- 1/15 JR東日本 電車代 1,200円 現金
- 1/20 NTTドコモ 携帯代 10,000円 普通預金（事業80%按分）
- 1/25 スターバックス 打合せ 1,500円 現金
- 1/31 クライアントA システム開発 500,000円 売掛金

終わったら1月の試算表を見せてください。
```
